# 💪 Качалка Кликер — Telegram Mini App

## Быстрый старт

```bash
# 1. Создай .env
cp .env.example .env
# вставь BOT_TOKEN и GAME_URL

# 2. Запуск
docker compose up -d
```

Игра на `http://сервер:8080` · Бот слушает Telegram

---

## Переменные окружения

| Переменная | Описание |
|---|---|
| `BOT_TOKEN` | Токен от @BotFather |
| `GAME_URL` | HTTPS-адрес игры (обязателен для Telegram) |
| `PORT` | Порт веб-сервера (по умолчанию 8080) |
| `DB_PATH` | Путь к SQLite базе лидерборда |

## HTTPS бесплатно
```bash
cloudflared tunnel --url http://localhost:8080
```

## Без Docker
```bash
pip install -r requirements.txt
python main.py
```

- ⚡ Энергия — восстанавливается автоматически

- 👑 Таблица лидеров — пьедестал топ-3 + таблица до топ-20

- 🔥 Престиж — множитель ×1.5, ×2.0, ×2.5...
